🌟 Tribute Page – Dr. A. P. J. Abdul Kalam

📌 Project Overview
This is a simple Tribute Page created using HTML and CSS to honor Dr. A. P. J. Abdul Kalam, the former President of India and one of the country's most respected scientists. The page showcases his name, lifespan, image, a famous quote, and a brief biography in a clean and responsive layout.
This project was developed as part of my journey in learning web development and strengthening my understanding of HTML and CSS fundamentals.

🚀 Features
Responsive and clean webpage layout
Separate HTML and CSS files
Centered heading with Dr. Kalam's name and lifespan
Image displayed alongside his famous quote
About section with a short biography
Hover effect on the image
Simple footer
Beginner-friendly code structure

🛠️ Technologies Used

CSS3
