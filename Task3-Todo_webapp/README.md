**To-Do List Web App**

A simple and interactive To-Do List Web Application built using HTML, CSS, and JavaScript. This project helps users manage their daily tasks by allowing them to add, edit, complete, and delete tasks. It also separates tasks into Pending Tasks and Completed Tasks for better organization.

**Features**
Add new tasks
Edit existing tasks
Mark tasks as completed
Separate Pending and Completed task lists
Delete tasks from either list
Display the date and time when a task is added
Display the date and time when a task is completed
Responsive and user-friendly interface

**Technologies Used**
HTML5 – Structure of the web application
CSS3 – Styling and responsive layout
JavaScript (ES6) – Task management and interactivity

**How to Run the Project**
Download or clone this repository.
Open the project folder.
Open index.html in any modern web browser.
Start adding and managing your daily tasks.
No additional software or dependencies are required.

**How to Use**
Enter a task in the input field.
Click the Add Task button to add it to the Pending Tasks list.
Use the Complete button to move a task to the Completed Tasks list.
Use the Edit button to modify a task.
Use the Delete button to remove a task from the list.
View the timestamps showing when tasks were added and completed.
